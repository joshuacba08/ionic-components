# Componentes en Ionic +6

### El siguiente proyecto hace una recopilación de una gran cantidad de componentes de los cuales podemos hacer uso con Ionic 6+.

<br/><br/>

### **Instalaciones previas:**
#### Recordar para usuarios de Windows, abri la terminal como ***administrador*** y para usuarios de Linux, anteponer el comando "***sudo***".

- [Angular CLI](https://angular.io/cli):
  
  ```
  $ npm install -g @angular/cli
  ```
- [Ionic 6+](https://ionicframework.com/docs/intro/cli):

    ```
    npm install -g @ionic/cli
    ```

<br/><br/>


### **Creando nuestra primera app con Ionic 6+**
#### Para iniciar nuestra primera app en Ionic, usaremos el siguiente comando:
```
ionic start nombreDeMiApp
```
Vamos a tener la opción de elegir qué Framework usar, en este caso seleccionaremos ***Angular***

![Framework select](https://res.cloudinary.com/ionicimagesbank/image/upload/v1635794002/ionic/imagen_2021-11-01_161320_tjpv0b.png)

<br></br>

Podemos iniciar nuestro proyecto de Ionic con un template en particular:

![templates select](https://res.cloudinary.com/ionicimagesbank/image/upload/v1635799962/ionic/imagen_2021-11-01_175241_nnjpg3.png)

En este caso usaremos blank para iniciar nuestro proyecto desde cero, y así poder realizar las configuraciones que necesitemos. Sin embargo, puedes probar las plantillas que te interesen para conocer la estructura de cada tipo de aplicación.
